Screen Share

Node.js JavaScript WebRTC Socket.IO
Website GitHub

A simple screen-sharing website that allows users to easily share their screen with others in real time.

Key Features:

  • Secure screen sharing powered by WebRTC
  • Room-based session management for organized sharing
  • Optional random room join feature for some entertainment!

Future Plans:

  • Live Chat feature
  • Support for casting from mobile

실시간으로 화면을 다른 사람들과 쉽고 간단하게 공유할 수 있는 화면 공유 웹사이트입니다.

주요 기능:

  • WebRTC 기반의 안전한 화면 공유
  • 방 단위의 세션 관리
  • 랜덤 방 참가!

추후 개발 계획:

  • 실시간 채팅 기능
  • 모바일 화면 공유 지원
