Astronaut Shooter

Unity C#
GitHub YouTube Download

My first video game project — a fast-paced third-person shooting survival game where an astronaut avoids randomly spawned obstacles and a chasing robot while collecting hearts to stay alive.

The game is continuously updated with modernized code and enhanced mechanics.

Key Learnings:

  • Bullet shooting mechanics using raycasting for precise aiming
  • Object pooling to efficiently manage multiple spawned objects
  • OOP principles to enhance modularity and maintainability
  • Asset management to organize and optimize game resources

Future Plans:

  • Multiplayer support using Socket.IO or Unity Netcode
  • Weapon upgrade system
  • Diverse enemy types

저의 첫 게임 프로젝트로, 무작위로 생성되는 장애물과 추격하는 로봇을 피하면서 하트를 모아 생존하는 3인칭 슈팅 생존 게임입니다.

첫 프로젝트지만, 새로 배우는 개발 방식들을 지속해서 적용하여 개선하고 있습니다.

주요 학습 내용:

  • Raycasting 기반 총알 발사 메커니즘
  • 여러 객체를 효율적으로 관리하기 위한 오브젝트 풀링
  • 모듈성과 유지보수를 위한 객체지향 프로그래밍 원칙 적용
  • 게임 리소스의 정리 및 최적화를 위한 에셋 관리

추후 개발 계획:

  • Socket.IO 또는 Unity Netcode를 활용한 멀티플레이어 기능 구현
  • 무기 업그레이드 시스템 추가
  • 다양한 종류의 적 캐릭터 도입
