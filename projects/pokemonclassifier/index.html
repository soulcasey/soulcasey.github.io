Pokemon Classifier

포켓몬 머신러닝 분류

PyTorch Python CUDA
Dataset GitHub

Pokemon Classifier is a computer vision model developed as part of NJIT’s Machine Learning course. Trained on over 3,000 images of the original 151 first-generation Pokemon, the model identifies Pokemon in real time using a camera feed and classifies them by name.

포켓몬 머신러닝 분류는 뉴저지 공과대학교 머신러닝 수업의 마지막 과제로 개발된 컴퓨터 비전 모델입니다. 1세대 오리지널 포켓몬 151종의 이미지 3,000장 이상을 학습하여 카메라 피드를 통해 포켓몬을 실시간으로 인식하고 이름으로 분류합니다.
